Tabla de Etiquetas HTML

  • <h1>: Título Principal
  • <h2>: Subtítulo
  • <h3>: Subtítulo Secundario
  • <h4>: Subtítulo Menor
  • <h5>: Subtítulo Muy Menor
  • <h6>: Subtítulo Más Pequeño
Etiqueta Descripción Ejemplo
<p> Representa un párrafo de texto y se usa para separar bloques de texto de forma ordenada. <p>Este es un párrafo.</p>
<h1> a <h6> Son etiquetas de encabezado que sirven para definir títulos o subtítulos, donde <h1> es el más importante y grande, y <h6> el menos importante y más pequeño. <h1>Título Principal</h1>
<h2>Subtítulo</h2>
<h3>Subtítulo Secundario</h3>
<h4>Subtítulo Menor</h4>
<h5>Subtítulo Muy Menor</h5>
<h6>Subtítulo Más Pequeño</h6>
<strong> Resalta el texto como importante, generalmente en negrita, es usado para enfatizar palabras clave o partes importantes del contenido. <strong>Texto Importante</strong>
<em> Destaca el texto con énfasis, generalmente en cursiva para resaltar con un énfasis suave. <em>Texto Enfático</em>
<br> Insertar un salto de línea, se usa para romper una línea sin iniciar un nuevo párrafo. <p>Línea 1<br>Línea 2</p>
<hr> Inserta una línea horizontal y sirve para dividir contenido visualmente. <hr>

Tabla de Listas HTML

Etiqueta Descripción Ejemplo
<ul> Crea una lista donde los ítems no tienen orden específico usando viñetas. <ul>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
</ul>
<ol> Crea una lista numerada, donde cada ítem tiene un número o letra. <ol>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
</ol>
<li> Cada ítem dentro de una lista, ya sea ordenada o no ordenada y va dentro de las etiquetas <ul> o <ol>. <ul>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
</ul>
