Semana 1 - Fundamentos de Tecnologías Web

Soluciones web: Aprendí a diferenciar entre sistema web, aplicación web, sitio web y página web. Un sistema web es un conjunto de aplicaciones y servicios interconectados; una aplicación web es un software accesible desde el navegador; un sitio web es un conjunto de páginas relacionadas bajo un dominio; y una página web es un solo documento dentro de un sitio.

Tecnologías básicas: Estudié los lenguajes esenciales como HTML para la estructura, CSS para el diseño y JavaScript para la interactividad. También conocí la importancia de los gráficos y recursos multimedia en la web.

Funcionamiento de la web: Profundicé en cómo funciona Internet: el DNS traduce nombres de dominio a direcciones IP, el protocolo TCP/IP permite la comunicación entre dispositivos y el protocolo HTTP es el encargado de transferir información entre cliente y servidor.

Roles en el desarrollo: Analicé los distintos roles: el desarrollador frontend se encarga de la interfaz y experiencia de usuario; el backend gestiona la lógica, datos y servidores; y el fullstack domina ambos campos, permitiendo una visión integral del desarrollo.

Herramienta de desarrollo: Instalé y configuré Visual Studio Code, aprendiendo a usar sus funciones básicas como la edición de código, instalación de extensiones y gestión de proyectos, lo que facilita mucho el trabajo diario como desarrollador.

Semana 2 - HTML, XML y CSS

HTML y XML: Aprendí a construir páginas web usando HTML5, comprendiendo la estructura básica de un documento, el uso de etiquetas semánticas y la diferencia con XML, que es más estricto y usado para el intercambio de datos. Analicé el árbol DOM y cómo el navegador interpreta y renderiza el contenido.

Árbol DOM y Ciclo de Vida: Profundicé en el árbol de nodos (DOM), cómo se genera y manipula, y el ciclo de vida de una página web desde que se solicita hasta que se muestra completamente en el navegador.

SEO y Compatibilidad: Estudié técnicas para optimizar páginas en motores de búsqueda (SEO), como el uso correcto de etiquetas, atributos alt en imágenes y estructura semántica. También revisé la importancia de la compatibilidad entre navegadores y cómo probar el correcto funcionamiento en diferentes entornos.

CSS3 y Diseño: Aprendí la sintaxis de CSS3, la diferencia entre display block, inline e inline-block, y el uso de unidades absolutas y relativas. Practiqué con flexbox y grid layout para crear diseños modernos y responsivos, y entendí cómo aplicar posiciones, márgenes y paddings para lograr un diseño fluido y adaptable a cualquier dispositivo.

Semana 3 - Bootstrap y Tailwind CSS

Bootstrap: Aprendí a utilizar Bootstrap para crear layouts responsivos rápidamente, aprovechando su sistema de grillas y componentes predefinidos como botones, tarjetas, formularios y menús de navegación. Profundicé en la personalización de estilos y la integración de utilidades para mejorar la experiencia de usuario y la accesibilidad.

Tailwind CSS: Descubrí el enfoque utility-first de Tailwind, usando clases para controlar tipografía, colores, espaciados, bordes y efectos visuales. Experimenté con flexbox, grid y la creación de componentes UI reutilizables, logrando diseños modernos y adaptables. Practiqué la composición de layouts complejos y la adaptación a diferentes dispositivos.

Comparación: Analicé las ventajas y diferencias entre Bootstrap y Tailwind, entendiendo cuándo conviene usar cada uno según el tipo de proyecto, la necesidad de personalización y la velocidad de desarrollo.

Semana 4 - JavaScript y TypeScript

JavaScript: Profundicé en los motores de JavaScript, tipos de datos (números, strings, objetos, booleanos), operadores, estructuras de control y bucles. Aprendí diferentes formas de declarar funciones (normales, flecha, anónimas, closures) y cómo se invocan. Practiqué la manipulación del DOM para modificar elementos y estilos desde JavaScript.

TypeScript: Conocí las ventajas de TypeScript sobre JavaScript, como el tipado estático y la detección temprana de errores, lo que ayuda a desarrollar aplicaciones más robustas y mantenibles.

Canvas y animaciones: Exploré el uso de canvas para crear animaciones y gráficos dinámicos, mejorando la interactividad de las páginas web. Aprendí a dibujar formas, trabajar con coordenadas y crear animaciones básicas usando JavaScript.

Semana 5 - Framework JS y Componentes

Framework JS: Aprendí cómo funcionan los frameworks modernos de JavaScript (como React), su estructura basada en componentes y el concepto de client side rendering para mejorar la experiencia de usuario.

Gestión de dependencias: Usé herramientas como npm/yarn para instalar librerías y gestionar proyectos. Practiqué la creación de proyectos desde cero y la organización del código en módulos.

Componentes y JSX: Profundicé en el desarrollo basado en componentes, el uso de JSX para escribir código similar a HTML dentro de JavaScript, y el manejo de props y children para pasar información entre componentes.

Estilos: Experimenté con diferentes formas de aplicar estilos: inline, hojas de estilos, módulos CSS y styled components, entendiendo sus ventajas y desventajas.

Semana 6 - Eventos, APIs y Routing

Eventos y Renderizado: Aprendí a manejar eventos en frameworks JS, como clicks y cambios en formularios. Profundicé en el renderizado condicional e iterativo para mostrar u ocultar elementos y listas dinámicamente.

Formularios y Routing: Practiqué la creación y validación de formularios, y la navegación entre diferentes vistas usando routing, mejorando la estructura de las aplicaciones.

Consumo de APIs: Estudié cómo consumir APIs externas usando promesas y async/await, y utilicé la librería Axios para obtener y mostrar datos en tiempo real, integrando información dinámica en mis proyectos.

Semana 7 - Hooks en Framework JS

Hooks básicos: Aprendí a usar los hooks principales de React como useState para manejar estados, useEffect para gestionar efectos secundarios, y useContext para compartir información entre componentes sin necesidad de props.

Hooks avanzados: Profundicé en hooks como useRef para referencias, useReducer para manejar estados complejos, useCallback y useMemo para optimizar el rendimiento de las aplicaciones.

Hooks personalizados: Descubrí cómo crear mis propios hooks personalizados para reutilizar lógica y mejorar la organización del código en proyectos grandes.
